
2 - Processes

Creating a process:
    pid_t pid = fork();
creates a child process
returns 0 to the child, the created process id to the parent, -1 if failed
parent runs before child

Waiting for a process:
    pid_t pid = wait(NULL);
blocks parent until child finishes
returns id of terminated child, or -1 if failed
returns immediately if a child has lready finished

Process ids:
    pid_t getpid(void); //returns current process id
    pid_t getppid(void); //returns parent id
    pid_t waitpid(pid_t pid, int *status, int options); //waits for a specific child process
if called with -1 will wait for any child

─────────ˏˋ ↳  3 - Signals ⸙ ˊˎ─────────

Signal handling:
    int sigaction(int signum,                               // signal number
        const struct sigaction *_Nullable restrict act,     // pointer to action to take
        struct sigaction *_Nullable restrict oldact);       // previous action

Example:
    #include <signal.h>

    struct sigaction act;

    void handler(int signum) {
        printf("Caught signal %d\n", signum);
    }

    int main () {
        act.sa_handler = handler;
        sigaction(SIGINT, &act, NULL);
    }

─────────ˏˋ ↳  5 - File Descriptors and I/O ⸙ ˊˎ─────────

Opening a file:
    int fd = open("my_file.txt", O_RDONLY);
returns integer file descriptor (fd)
alternatives: O_WRONLY, O_CREAT, O_TRUNC
can also combine as O_CREAT | O_WRONLY

Writing to the file:
    char buffer[] = "Hello, world!";
    write(fd, buffer, sizeof(buffer));
returns number of bytes written

Reading from a file:
    char buffer[1024];
    read(fd, buffer, sizeof(buffer));
returns number of bytes read

Closing a file:
    close(fd);
returns 0

all return int, == -1 if error
